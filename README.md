# questionarioCpa

Sistema web para aplicação e gestão dos questionários da **Comissão Própria de Avaliação (CPA)** da UEAP.

## Overview do sistema

Este projeto foi desenvolvido para centralizar a avaliação institucional, permitindo a coleta estruturada de respostas e a consolidação dos resultados para apoio à tomada de decisão.

### 1. CPA

A CPA é responsável por conduzir os processos de autoavaliação institucional. No sistema, esse fluxo é suportado por funcionalidades que permitem:

- disponibilizar avaliações de forma organizada;
- registrar participação dos públicos avaliadores;
- manter histórico dos ciclos de avaliação.

### 2. Questionários e dimensões

O núcleo do sistema está na gestão dos questionários, estruturados por **dimensões avaliativas**. Isso possibilita:

- criação e edição de perguntas por dimensão;
- organização temática para facilitar análise dos resultados;
- aplicação padronizada para diferentes públicos e períodos.

### 3. Relatório

Após a coleta das respostas, o sistema oferece base para geração de relatórios de avaliação, com foco em:

- consolidação dos dados por dimensão;
- visualização de indicadores de desempenho institucional;
- suporte à elaboração de diagnósticos e planos de melhoria.
